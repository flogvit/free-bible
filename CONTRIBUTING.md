# Contributing to Free Bible

Thank you for your interest in contributing to Free Bible! This project aims to provide free Bible translations in as many languages as possible.

## How to Contribute

### Reporting Bugs

If you find a bug, please open a [GitHub Issue](https://github.com/flogvit/free-bible/issues/new?template=bug_report.md) with:

- A clear description of the problem
- Steps to reproduce the issue
- Expected vs actual behavior
- Which translation/language is affected (if applicable)

### Suggesting Features

Have an idea for a new feature or improvement? Open a [Feature Request](https://github.com/flogvit/free-bible/issues/new?template=feature_request.md).

### Translation Corrections

If you find errors in a Bible translation, please open an issue with:

- The translation code (e.g., OSNB1, OSNB2)
- Book, chapter, and verse reference
- The current text and suggested correction
- Explanation of why the correction is needed

### Code Contributions

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Test your changes locally
5. Commit with clear messages
6. Push to your fork and open a Pull Request

### Adding New Languages

If you want to help translate the Bible into a new language, please open an issue to discuss the language and approach before starting work.

## Development Setup

See the [README](README.md#developer) for development setup instructions.

## Code Style

- JavaScript/Node.js for generation scripts
- Keep scripts consistent with existing patterns
- Use meaningful variable and function names
