# Høysangen

**Om boken:** Høysangen, også kjent som Salomos Høysang (שִׁיר הַשִּׁירִים), er en samling kjærlighetsdikt som tilskrives kong Salomo. Boken skildrer kjærligheten mellom en brudgom og hans brud gjennom rik poetisk symbolikk med bilder fra naturen, hager og vingårder.

**Hovedtema:** Kjærlighetens skjønnhet, kraft og uovervinnelighet mellom mann og kvinne.

**Innholdsoversikt:**
- **Kjærlighetens begynnelse og forelskelse** (Kapittel 1-2:7)
- **Kjærestenes gjensidige beundring og lengsel** (Kapittel 2:8-3:5)
- **Brudens skjønnhet og brudgommens lovprisning** (Kapittel 3:6-5:1)
- **Adskillelse, savn og søken** (Kapittel 5:2-6:3)
- **Gjenforening og fornyet kjærlighet** (Kapittel 6:4-8:4)
- **Kjærlighetens segl og evige kraft** (Kapittel 8:5-14)

**Nøkkeltemaer:**
- **Gjensidig kjærlighet:** Brudgommen og bruden uttrykker dyp hengivenhet og beundring for hverandre gjennom hele boken.
- **Naturens symbolikk:** Hager, vingårder, blomster og dyr (gaseller, duer) brukes som bilder på kjærlighet og skjønnhet.
- **Kjærlighetens styrke:** Kjærligheten beskrives som sterk som døden, med flammer som vann ikke kan slukke (8:6-7).
- **Lengsel og søken:** Adskillelse og gjenforening viser kjærlighetens dybde og utholdenhet.
- **Jerusalems døtre:** Et tilbakevendende kor som vitner om kjærligheten og mottar edformler om ikke å vekke kjærligheten før den selv vil.
